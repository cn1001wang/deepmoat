## Q-0002 云服务器选型

- **日期**:2026-06-18
- **状态**:open
- **阻塞任务**:T2.1 / T2.2 / T2.4(P2A 云部署)
- **背景**:用户已确认要部署到云服务器,但没说具体平台 / 配置 / 域名
- **决策维度**:
  - 平台:阿里云轻量 / 腾讯云轻量 / Hetzner / DigitalOcean / 自有 NAS
  - 规格:2C4G(够用)vs 4C8G(舒适);PG 数据量 < 10GB,IO 不大
  - PG 自托管 vs RDS:自托管便宜 80%,但要自己备份;RDS 省心,但成本高
  - 域名:国内备案 vs 海外不备案
  - 流量限制:用户单人用,流量极小
- **agent 倾向**:**Hetzner CX22(2vCPU 4GB,€4.5/月)+ 自托管 PG + Cloudflare 域名(DNS+CDN+免备案)**;成本最低、性能够用、CN 访问可用 cloudflare warp 兜底。如果对国内访问速度敏感,改阿里云轻量香港版(无需备案,30/月)
