【巴菲特开场：公司基本信息】
1. 公司如何赚钱（一句话）
华测导航靠“高精度定位装备 + 行业系统应用解决方案”卖软硬件与服务，在测绘地理信息、农机自动驾驶、建筑与位移监测等场景持续变现。

2. 收入结构与核心业务占比
- 财报日期：2025-12-31（2026-04-17披露年报）。
- 2025年营收约36.99亿元，归母净利约6.85亿元。
- 分产品（2025）：系统应用及解决方案约20.15亿元（约54.5%）；高精度定位装备约16.85亿元（约45.5%）。
- 分地区（2025）：中国大陆约24.32亿元（约65.7%）；海外约12.67亿元（约34.3%）。

3. 行业位置与主要对手
- 行业定位：高精度卫星导航（GNSS）与行业数字化应用厂商。
- 主要可比：中海达、司南导航、海格通信等（侧重场景与业务重叠部分）。
- 相对优势：公司已形成“设备 + 解决方案 + 渠道”的组合，业务不只卖单一硬件。

4. 近4-5年增长与盈利趋势
- 2021-2025：营收19.03亿 -> 36.99亿，4年CAGR约18.1%；归母净利2.94亿 -> 6.85亿，4年CAGR约23.5%。
- 毛利率2021-2025由54.6%抬升到61.0%，净利率由15.2%升至18.5%，盈利质量持续改善。

5. 资产负债与现金流安全性
- 2025年末资产负债率约30.5%，杠杆可控。
- 2025年经营现金流约6.99亿元，自由现金流约2.46亿元，连续为正。
- 需盯住营运资本：应收账款/收入约30.4%（近年在29%-33%区间），现金回款管理是关键。

【投资者问答：沃伦与查理】
投资者：这家公司到底是“设备公司”还是“解决方案公司”？
沃伦：它已经不是单纯硬件厂。收入里过半来自系统应用及解决方案，这意味着它在往“场景化价值”迁移，不只是拼设备参数。
查理：方向对，但别太乐观。解决方案业务一旦规模上来，项目管理、交付效率、应收回款会成为新的风险点。

投资者：它的护城河到底在哪里？
沃伦：护城河有三层。第一层是高精度定位算法与产品可靠性；第二层是行业场景Know-how；第三层是渠道和客户粘性。硬件可被追赶，但“设备+场景+服务”的组合更难一次性复制。
查理：护城河还不算特别宽。这个行业技术迭代快、价格竞争也在，若产品更新慢半拍，毛利率改善会被迅速吃掉。

投资者：过去几年利润增长很好，这种趋势能持续吗？
沃伦：从2021到2025的复合增速看，增长具备延续性，不是只靠一次性项目。毛利率和净利率提升说明效率在改善。
查理：增长能不能延续，要看两个先行指标：应收周转和海外扩张质量。如果应收继续抬高而现金流跟不上，利润含金量会打折。

投资者：海外占比已经三成多，这是机会还是风险？
沃伦：是机会。海外收入占比34%说明公司不只靠国内市场，抗单一市场波动能力更强。
查理：也是风险。海外增长往往伴随渠道建设成本、合规成本和汇率波动。只看收入增速不够，要看海外净利润和回款是否同步。

投资者：按现在价格，值不值得长期拿？
沃伦：这更像“中速成长 + 稳步兑现”的公司，39倍PE并不便宜，但在其历史分位也不算离谱。
查理：核心判断不是“贵不贵”三个字，而是“未来两年利润兑现能不能匹配估值”。若增速降到个位数，估值会被压缩。

本杰明：

- 时间锚：价格日2026-04-17；财报日2025-12-31。

2. 经营现金流是否持续覆盖利润增长（避免“利润增长、现金不增长”）。
- 若后续兑现稳健，当前估值可被消化；若增速和现金流同步下台阶，估值中枢会下移。

【综合收束】
1. 华测导航是“从硬件向解决方案升级”的成长型公司，过去5年财务兑现较好。
2. 当前最值得肯定的是：收入和利润双增长、毛利率提升、海外占比提升。
3. 当前最该警惕的是：应收占比偏高、项目化业务带来的回款与交付风险、行业竞争导致的价格压力。
4. 这只票的核心跟踪框架是三条线：
- 增长线：收入与净利增速是否仍双位数；
- 质量线：经营现金流与自由现金流是否跟上；
- 估值线：在39倍PE附近，增长兑现是否匹配估值溢价。
