【巴菲特开场：公司基本信息】
1. 公司如何赚钱（一句话）
海天味业靠“高频刚需调味品（酱油/蚝油/酱类）+ 全国经销网络 + 规模制造”赚取稳定现金流。

2. 收入结构与核心业务占比（海天 vs 卡夫亨氏）
- 海天味业（2025年报，报表日 2025-12-31）：营收 288.73 亿元。按产品看：酱油约 51.7%，蚝油约 16.9%，酱类约 10.1%，其他主营约 16.2%。
- 卡夫亨氏（2025 10-K，报表日 2025-12-27）：净销售额 249.42 亿美元。按区域看：北美 185.86 亿（约 74.5%）、国际成熟市场 35.39 亿（约 14.2%）、新兴市场 28.17 亿（约 11.3%）。

3. 行业位置与主要对手
- 海天：A股调味品龙头，核心对手包括中炬高新（厨邦）、千禾味业、李锦记等。
- 卡夫亨氏：全球包装食品巨头之一，对手包括雀巢、百事食品、通用磨坊、亿滋及零售商自有品牌。

4. 近4-5年增长与盈利趋势
- 海天（2021-2025）：营收 250.04 亿 -> 288.73 亿，4年CAGR约 3.7%；归母净利 66.71 亿 -> 70.38 亿，4年CAGR约 1.4%。2022-2023承压后，2024-2025恢复增长。
- 卡夫亨氏（2022-2025）：净销售 264.85 亿美元 -> 249.42 亿美元，连续回落；2025年因减值影响出现净亏损（-58.48亿美元），但经营现金流仍为正。

5. 资产负债与现金流安全性
- 海天：2025年经营现金流 77.46 亿元；资产负债率 19.82%；货币资金 246.04 亿元，财务安全垫较厚。
- 卡夫亨氏：2025年经营现金流 44.62 亿美元；有息债务约 212.19 亿美元（流动+长期），净债务约 175.44 亿美元，杠杆显著高于海天。

6. 当前估值位置（价格日期：2026-04-17）
- 海天：收盘价 39.94 元，PE(TTM) 33.21，PB 5.91；按2021-2026区间，PE约在12.8%分位、PB约在8.2%分位，处在近年偏低估值带。
- 卡夫亨氏：股价约 22.64 美元，市值约 262 亿美元；GAAP口径因2025年减值为亏损，静态P/E失真，需更多看现金流与归一化盈利。

【投资者问答：沃伦与查理】
投资者：海天会不会像卡夫亨氏一样，最终陷入“只能靠打折维持份额”的价格战？
沃伦：会有价格竞争，但两家公司所处赛道不完全一样。海天的核心品类是厨房刚需高频消耗品，品牌习惯和复购黏性更强。卡夫亨氏更多暴露在成熟包装食品赛道，零售商集中度高、自有品牌替代更强，价格战传导更快。
查理：别把“高频”误读成“无敌”。如果渠道库存错配、终端促销常态化，调味品也会被迫让利。海天要避免走向卡夫那条路，关键不是喊品牌，而是持续产品迭代和渠道控制，不然毛利也会被磨薄。

投资者：那海天的护城河和卡夫亨氏相比，谁更硬？
沃伦：海天的护城河更偏“本土渠道深度 + 消费心智 + 规模制造效率”；卡夫亨氏更偏“全球品牌资产 + 分销体系”。但近几年卡夫的品牌溢价被零售和自有品牌持续挤压，护城河变窄更明显。
查理：卡夫的问题是“品牌在，议价权变弱”；海天要警惕的是“品牌在，增长变慢”。两者都不是会自动加深的护城河，必须靠资本配置和经营质量续命。

投资者：从财务体质看，哪家更能扛周期？
沃伦：海天明显更轻杠杆，资产负债率不到20%，账上现金很厚，抗波动能力强。卡夫的现金流也不错，但债务负担重、减值冲击大，容错率更依赖经营稳定。
查理：财务结构决定心理边界。净现金公司可以等，净负债公司必须跑。海天出错可以慢修，卡夫出错往往要更快出手。

投资者：海天过去几年利润增速不快，这是不是“成熟见顶”的信号？
沃伦：我更愿意把它看成“从高增长切到稳增长”。2022-2023是压力期，2024-2025已经恢复。对这种公司，重点看ROE质量、现金流和份额，而不是追逐高弹性。
查理：如果未来两三年只能靠提价而没量，或者只能靠促销保量，那就会向卡夫的问题靠近。你要盯的是“量价结构”，不是单看利润表一个数字。

投资者：如果二选一，海天和卡夫谁更像“能长期拿着”的标的？
沃伦：以当前财务和商业质量看，海天更像“高确定性、低杠杆、慢复利”；卡夫更像“低估值、现金流型、需要执行修复”的标的。
查理：一句话，海天是“贵一点买确定性”，卡夫是“便宜一点买修复难度”。你先决定自己要哪一种风险。

【本杰明插话：估值校准】
本杰明：先把尺子统一。
- 海天：用A股消费龙头常见口径看，当前PE 33倍不算便宜绝对值，但放在它自己近5年分位是偏低区间；若未来利润维持中低个位数增长，估值更多靠“稳定性溢价”而非高增长重估。
- 卡夫：GAAP利润受减值严重扰动，直接看PE会失真。更适合看“经营现金流/市值”和债务约束下的分红与回购能力。
- 对比结论：海天估值锚在“质量+确定性”，卡夫估值锚在“修复+现金回收”。前者怕增长下台阶，后者怕修复失败与价格战深化。

【综合收束】
1. 你问“海天会不会陷入卡夫式价格战”，答案是：有风险，但当前路径还不一样。
2. 海天当下更强的是财务韧性和本土渠道深度；卡夫当下更强的是现金流规模和低估值属性。
3. 真正的分水岭在未来8个季度：
- 海天看“销量恢复是否持续、毛利率是否稳定在较高区间、渠道折扣是否恶化”；
- 卡夫看“销量能否止跌、减值后经营质量是否改善、杠杆是否继续下行”。
4. 如果你偏好长期确定性，海天逻辑更顺；如果你偏好低估值反转，卡夫更像修复交易。
