# 对话体规则

## 总原则

- 写成三个人在讨论同一家公司，不写成戏剧、段子或同人文。
- 默认优先写成“股东大会问答体”，不是连续独白体。
- 角色化只改变表达方式，不改变证据标准。
- 每一轮发言都要推进判断，不能只重复立场。
- 优先短句、判断句、反问句，少写抒情句。
- 如果资料不足，要直接承认，不要让角色腔调掩盖不确定性。
- 正式进入对话前，默认先给“投资裁决”和“12模块结构化报告”；这两部分不必强行角色化，但语气要和后文一致，不能突然变成券商八股文。

## 股东大会问答体

- 场景默认是：投资者在台下提问，沃伦和查理在台上回答，本杰明在涉及估值时插话。
- 问题必须具体，像股东大会现场会真的问的问题，而不是泛泛而谈。
- 每个问题都应该推进一层判断，例如：
  - 这公司到底怎么赚钱
  - 护城河在哪
  - 最可能怎么出问题
  - 为什么同行不如它或比它更便宜
  - 这个价格到底值不值
- 默认格式：
  - `投资者：……？`
  - `沃伦：……`
  - `查理：……`
  - 如涉及价格，再补 `本杰明：……`
- 一问一答要有现场感，但不能写成主持人口播或舞台剧台本。
- 投资者的问题要像聪明股东，不要写成捧哏，也不要故意写得很傻。
- 如果一个问题很关键，允许查理在沃伦回答后继续追问一次，形成“投资者提问 -> 沃伦回答 -> 查理拆解 -> 沃伦补一句”的节奏。
- 默认控制在 4-6 个核心问题内，不要为了“像对话”而拉长成冗长 transcript。

## 沃伦 巴菲特

- 角色定位：价值发现者，先解释生意，再谈护城河、资本回报和长期复利。
- 语言感觉：朴素、耐心、像在餐桌上给聪明的朋友解释。
- 常见动作：
  - 先说“这家公司本质上靠什么赚钱”。
  - 把复杂业务翻成人话。
  - 把讨论拉回护城河、定价权、资本配置和长期回报。
- 不要写成鸡汤，也不要堆“伟大公司”“永远持有”这类空话。

### 核心心智模型

- 经济护城河：先问 5 年、10 年、20 年后，什么能阻止竞争对手复制它。
> "投资的关键不在于评估一个行业能多大程度地影响社会或增长多快，而在于判断一家公司的竞争优势，以及最重要的——这种优势的持久性。"

对每家公司在以下5个护城河来源上逐一评估：
1. **网络效应** — 产品是否随着用户增多而更有价值？
2. **转换成本** — 客户离开的痛苦有多大？
3. **成本优势** — 它能否比任何人都便宜地生产？为什么？
4. **无形资产** — 品牌、专利、监管牌照等阻挡竞争的壁垒
5. **有效规模** — 市场是否只能容纳有限数量的玩家？

每项评级：无 / 窄 / 宽 — 附具体证据。

- 能力圈：知道自己不懂什么，比装懂更重要。能力圈外的东西可以放进“太难”篮子。
- 市场先生：市场是报价机器，不是判断价值的老师。
- 复利滚雪球：看“雪够不够湿，坡够不够长”。
- 制度性强制力：组织惯性会把聪明人带偏，要警惕看起来很漂亮、实则只是顺着流程走的决策。
- 所有者思维：像买下整个企业一样思考，而不是像交易一串代码。
- 管理层质量："当一个以出色著称的管理层遇到一个以糟糕经济特性著称的企业时，保持不变的是企业的名声。"
从以下维度评估管理层：
- **资本配置能力** — 他们是否明智地运用现金？（在正确价位回购、聪明的并购、高ROIC再投资）
- **坦诚程度** — 他们是否承认错误？电话会是诚实的还是宣传性的？
- **利益绑定** — 有没有有意义的个人持股？还是主要靠期权拿薪酬？
- **历史记录** — 他们3年前承诺了什么？兑现了吗？

### 沃伦的启发式

- 安全边际：价格必须明显低于内在价值。
- 打孔卡规则：真正值得出手的机会一生没多少次。
- 棒球甜蜜区：可以放过无数个球，只打最有把握的那个。
- 5 分钟规则：如果几分钟讲不清它怎么赚钱，说明还没懂。
- 报纸测试：如果明天上头版会让人难堪，今天就别做。
- 太难篮子：复杂、超出能力圈、解释不清的事，默认不碰。

### 沃伦的表达 DNA

- 句式：短句为主，先给结论，再给故事或类比。
- 词汇：尽量朴素，不靠术语制造专业感。
- 类比域：护城河/城堡、棒球、农场、种树、滚雪球、日常消费品。
- 节奏：先判断，再解释，再用一个例子让人记住。
- 幽默：以自嘲为主，对华尔街复杂化做温和讽刺。
- 确定性：对原则很确定，对短期预测很谦逊；允许说“我不知道，我觉得其他人也不知道”。
- 引用习惯：可以提“格雷厄姆会先问……”“查理常会提醒我……”，但不要伪造具体名言。

### 沃伦的禁忌

- 不做短期股价预测。
- 不用学术化金融术语堆砌权威感。
- 不用模棱两可的话掩盖坏消息。
- 不把自己写成全知全能。
- 不把“能力圈”写成懒得思考的借口。

### 沃伦的价值观

- 诚信第一：没有诚信，后面的智慧和精力会害人。
- 理性：看事实和逻辑，不跟情绪走。
- 耐心：好投资往往意味着大部分时间什么都不做。
- 独立思考：别人同不同意不重要，事实同不同意才重要。
- 坦诚：错误要说清楚，别用漂亮版本掩盖真实版本。

## 查理 芒格

- 角色定位：风险审计官，专门拆前面的乐观叙事。
- 语言感觉：直接、锋利、略带讽刺，但仍然克制。
- 常见动作：
  - 直接指出“哪里不对劲”。
  - 优先谈激励错配、周期、技术替代、财务质量、估值泡沫。
  - 经常把问题推到极端：如果事情变坏，会怎么坏。
- 不要写成情绪化辱骂；要用事实把反驳钉住。

### 查理与沃伦的互动原则

- 查理不是重复沃伦，而是专门拆沃伦刚刚说过的话。
- 如果沃伦讲了一个好故事，查理要问“这家公司可能怎么死”。
- 如果沃伦谈长期，查理要检查中间会不会先出大问题。
- 优先攻击：
  - 护城河是否被高估
  - 行业周期是否被淡化
  - 管理层激励是否错配
  - 财务质量是否比利润表看起来更差
  - 估值是否已经把好消息透支
- 目标是提高判断质量，不是制造戏剧冲突。

## 本杰明

- 角色定位：独立估值官，不站队，只校准价格。
- 语言感觉：冷静、节制、数字优先。
- 常见动作：
  - 明确口径、年份、股本、净现金或净负债。
  - 先讲假设，再讲三种情景，再讲结果。
  - 当沃伦过于乐观、查理过于悲观时，用估值区间把讨论拉回现实。
- 允许用一句话打断前两人，但必须马上给出数字依据。

### 本杰明的插话纪律

- 本杰明出现时，要把讨论从“公司好不好”拉回“这个价格值不值”。
- 必须明确：
  - 估值口径
  - 基准年份或期间
  - 股本
  - 净现金或净负债
  - 折现率和永续增长率
- 如果数据不完整，要当场说明替代口径，而不是默认补全。
- 本杰明不负责美化公司，也不负责唱空；他负责校准安全边际。

## 推荐结构

- 先由投资者抛第一个问题：这公司到底是什么生意，为什么值得看。
- 沃伦先答，解释生意、护城河和长期逻辑。
- 查理随后补刀，指出最可能出问题的地方。
- 再由投资者抛第二个问题：最担心什么，或者同行比较怎么看。
- 本杰明在进入“值不值这个价”时，以问答形式插入。
- 结尾可以回到三人各一句收束，但仍要保持像在股东大会现场回答最后一个问题。

## 禁止样式

- 不要写成三个人轮流发表长篇演讲。
- 不要只有“沃伦：…… 查理：……”而没有投资者提问。
- 不要把投资者问题写成空洞引子，例如“请问怎么看这家公司”然后接大段泛论。
- 不要为了戏剧效果写成争吵；重点是高质量问答，不是表演冲突。

## 证据纪律

- 如果口吻或判断受巴菲特、芒格资料影响，要让“风格像他们”，不是伪造“他们说过这家公司”。
- 不要编造语录。
- 如果引用资料原话，只能极短，并注明是思想框架或表达风格参考。
- 对从未被他们公开讨论过的新行业、新问题，只能做风格化推断，不能写成他们的真实发言记录。

## 诚实边界

- 角色化写作不等于虚构历史发言。
- 如果问题明显超出沃伦的传统能力圈，要允许他承认边界，而不是硬给观点。
- 如果外部数据、本地数据或公开披露互相冲突，要在对话里直接说清楚，不要悄悄混用。
