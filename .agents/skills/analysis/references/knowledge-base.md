# 知识库使用规则

`agents/` 目录保留原始资料，不再作为行为入口。

## 可引用资料

- 巴菲特相关资料：`agents/巴菲特知识库/`
- 芒格相关资料：`agents/芒格知识库/`

## 何时读取

- 用户明确要求“用巴菲特/芒格视角”时再打开对应资料。
- 用户要求“像他们对话”“像股东信口吻”“像芒格直接反驳”时，读取最相关的少量文档，不要批量加载。
- 资料用于补充框架和措辞风格，不用于替代项目内财务数据。

## 引用原则

- 优先提炼观点，不长篇复制原文。
- 如果引用原话，保持极短并注明是观点引用。
- 观点冲突时，以当前财务与业务事实为准。
- 目标是让人物气质和思考方式接近资料，不是伪造他们对现实公司的具体发言。

## 与数据 skill 的配合

- 外部 `tushare-data`：负责 Tushare 官方数据、宏观、行业、公告等通用取数。
- 本仓库 `deepmoat-local-data`：负责 `app/models/models.py` 对应的本地数据库、本地 HTTP 接口和 `outputs/` 落盘。
- 做公司分析时，优先把两者结合，而不是让本仓库 skill 冒充外部 `tushare-data`。
