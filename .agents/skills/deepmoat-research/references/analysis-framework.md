# 分析框架口径

> 对齐投资体系 v2 第五~八章。供深度估值模式与对话模式判断时引用的阈值与口径；结构化研报模式也参照本框架组织证据。

核心理念：**利润表可以讲故事，现金流量表更接近真相。** 现金流 > 利润 > 营收。

## 一、三表分析

### 1.1 资产负债表

- 流动比率 ≥ 2（保守）；< 1 警惕短期偿债压力
- 货币资金 vs 短期借款：覆盖倍数 ≥ 1.5
- 应收账款 / 存货：增速不应持续高于营收增速
- 商誉占净资产：< 20% 健康；> 30% 高风险
- 资产质量：有形资产占比、可变现性

### 1.2 利润表

- 5-10 年持续盈利
- 毛利率 / 净利率稳定或上行
- 扣非净利润 = 净利润的可信度（差距 > 30% 警惕）
- 非经常性损益占比

### 1.3 现金流量表（最重要）

| 指标 | 健康 | 优秀 | 警惕 |
|-----|-----|-----|-----|
| 经营现金流净额 | 5 年大部分为正 | 5 年全为正 | 连续 2 年为负 |
| 经营现金流 / 净利润 | 5 年累计 > 0.8 | > 1 | < 0.6 |
| 销售商品收到的现金 / 营业总收入 | > 0.8 | > 1.1 | < 0.6 |
| 自由现金流 | 长期为正 | 长期 + 增长 | 长期为负 |
| 资本开支 / 经营现金流 | 60%-100% 正常 | < 60% 轻资产高质量 | > 100% 激进 |
| 分红来源 | 经营现金流 | 经营现金流且 FCF 仍正 | 借钱分红 |

## 二、成长能力

### 2.1 增长来源拆解（必做）

```
营收增长 = 量增 + 价增
        = (客户数变化 + 客单价变化) × 渗透率变化 × 市占率变化
```

| 拆解维度 | 关注问题 | 数据来源 |
|---------|---------|---------|
| 量增 | 出货量 / 用户数 / 客户数趋势 | 季度财报 + 行业数据 |
| 价增 | 单价变化（提价能力） | 季度财报 + 招股说明书 |
| 渗透率 | 行业整体渗透率所处阶段 | 行业研究报告 |
| 市占率 | 公司在行业中位置 | 第三方咨询（Wind/灼识/中商等） |
| 海外占比 | 是否打开第二增长曲线 | 公司年报 / 公告 |

### 2.2 增长可持续性 4 问

1. 行业天花板还有多远（渗透率 / 单价 / 用户基数）
2. 竞争格局变化方向（集中度上升/下降？份额来源？）
3. 商业模式可复制性（老业务的成功能复制到新业务吗？）
4. 关键依赖（单一大客户？单一大产品？单一国家？）

### 2.3 关键指标

| 指标 | 健康线 | 优秀线 | 警惕线 |
|-----|-------|-------|-------|
| 营收 5 年 CAGR | > 8% | > 15% | < 5% 或负增 |
| 扣非净利 5 年 CAGR | > 8% | > 15% | 持续低于营收增速（降利率） |
| 经营现金流 5 年 CAGR | > 8% | 同步营收 | 显著低于营收（应收暴增） |
| 单位经济（单店/单W/单ARPU） | 稳定或上行 | 持续优化 | 持续恶化 |
| 海外营收占比变化 | 上升 | 高位稳定 | 下降但管理层不解释 |

### 2.4 生命周期判断

| 生命周期 | 关注重点 | 估值方法 | 态度 |
|---------|---------|---------|------|
| 早期 | 赛道规模 / 团队 / 单位经济模型 | PS / EV/Sales | 仅 AI Agent 板块小仓位试水 |
| 成长期 | 市占率 / 复购 / 扩张速度 | PEG / DCF / EV/EBITDA | A 档主力 |
| 成熟期 | 价格能力 / 盈利质量 / 分红 | PE / DCF / 股息 | A 档主力 |
| 衰退期 | 不投 | 不投 | 拉黑 |

## 三、护城河分析

### 3.1 五大护城河类型（Morningstar 框架）

| 护城河类型 | 本质 | 典型行业 | 代表公司 |
|----------|------|---------|---------|
| 无形资产 | 品牌 / 专利 / 牌照 / 政府准入 | 高端消费、医药、烟酒 | 茅台、恒瑞 |
| 转换成本 | 切换成本太高 | 企业软件、银行账户、医疗设备 | 微软 Office、Bloomberg |
| 网络效应 | 用户越多越值钱 | 平台、社交、支付 | 微信、淘宝 |
| 成本优势 | 规模 / 技术 / 区位低成本 | 电池、化工、零售 | 宁德、好市多 |
| 有效规模 | 市场只能容纳少数玩家 | 区域公用事业、机场港口 | 上港集团 |

### 3.2 宽 / 窄护城河判定

| 等级 | 含义 | 判定标准 |
|------|------|---------|
| 宽护城河 | 未来 20 年仍能产生超额收益 | ROIC 长期 > WACC 5pct + 至少 2 种护城河来源 + 行业地位稳固 |
| 窄护城河 | 未来 10 年能产生超额收益 | ROIC 长期 > WACC + 1 种护城河来源 |
| 无护城河 | 行业平均回报 | ROIC ≈ WACC，无显著差异化 |
| 负护城河 | 长期亏损或低 ROIC | 应进入排雷 |

### 3.3 量化指标

| 指标 | 含义 | 阈值 |
|-----|------|------|
| ROIC | 投入资本回报率 | > WACC + 5pct = 宽护城河信号 |
| ROIC 持续性 | 近 10 年波动幅度 | 标准差 < 5pct = 稳定护城河 |
| 毛利率稳定性 | 近 10 年波动 | 标准差 < 3pct = 定价权强 |
| 净利率稳定性 | 近 10 年波动 | 标准差 < 2pct = 业务质量高 |
| 市占率趋势 | 近 5 年变化 | 上升或稳定 = 加固；下降 = 削弱 |
| 客户集中度 | TOP5 占比 | > 50% 加分（粘性）；> 80% 减分（依赖） |

### 3.4 不同行业看哪种护城河

| 行业 | 主要看 | 辅助看 |
|------|-------|-------|
| 消费（食品饮料/家电/服饰） | 品牌 + 渠道 | 转换成本（黏性 SKU） |
| SaaS / 企业软件 | 转换成本 + 网络效应 | 无形资产（数据沉淀） |
| 工业品 | 客户绑定 + 认证壁垒 + 成本优势 | 规模 |
| 储能集成 | 渠道 + 海外认证 + 品牌 | 技术（自研比例） |
| 储能 PCS/BMS/EMS | 技术 + 客户绑定 | 规模 |
| 游戏 | IP + 研发能力 + 用户社区 | 网络效应（玩家社交） |
| 公用事业 | 有效规模 | 政府准入 |

### 3.5 护城河变化信号（早期警告）

- 削弱信号：毛利率持续下行 / 市占率下行 / 大客户流失 / 主要专利到期
- 加固信号：毛利率提升 / 市占率提升 / 海外突破 / 标准制定参与权

### 3.6 真伪辨别（必做）

- 提价 5% 是否流失客户
- 客户是否对价格高度敏感
- 是否存在"非它不可"场景
- 替代品出现难度
- 更换供应商成本
- 结论：强 / 中 / 弱 / 伪护城河

## 四、估值方法选择

### 4.1 快速选择口诀

1. 现金流稳定 -> DCF / PE
2. 重资产 / 负债敏感 -> EV/EBITDA / PB
3. 亏损但收入增长快 -> PS / EV/Sales
4. 金融机构 -> PB + ROE / 剩余收益
5. 强周期行业 -> PB / EV/EBITDA（避开 PE）
6. 资产折价 / 困境反转 -> NCAV / 清算价值

### 4.2 能力圈对应估值方法

| A/A- 档方向 | 主估值 | 辅估值 | 安全边际锚点 |
|------------|-------|-------|-------------|
| 储能系统集成 | EV/EBITDA + DCF | PE（成熟期） | 单 W 净利历史中位数 ± 30% |
| 储能 EMS 软件 | PS -> EV/EBITDA | DCF | ARR 增长率 vs PS 隐含预期 |
| 储能 BMS/PCS | PE + PEG | EV/EBITDA | 毛利率历史区间 |
| 消费 | PE + DCF | 股息率 | 近 5 年 PE 分位 |
| AI Agent | PS / EV/Sales | DCF（成熟后） | 收入增速 vs PS 隐含预期 |
| 游戏（学习中） | PE + PEG | PS（早期） | 流水 / PS 历史区间 |

### 4.3 估值输出要求

- 多模型交叉：相对估值（PE/PB/PS/PEG/EV-EBITDA 历史分位）+ DCF（基准/乐观/悲观三情景）+ 反向 DCF（当前价隐含增长率是否现实）。
- 必须给数字区间，不可只给"高估/低估"。
- 安全边际判断：高估 / 合理 / 低估，并说明假设与置信度。
