# 评分模型（分模式，100 分制）

> 对齐投资体系 v2 第九章。筛选通过排雷与策略模板后，按公司特征选择评分模式。

## 模式选择决策树

```
公司基本面是否健康?
├── 健康(财务好+护城河) -> 是不是市场误判?
│   ├── 是 -> 【捡漏型评分】 (低估反转)
│   └── 否 -> 【稳健价值评分】 (慢慢拿)
└── 不健康但行业新 -> 是不是早期成长?
    ├── 是 -> 【高质量成长评分】 (PS 估值)
    └── 否 -> 不投
```

独立第四模式【护城河型评分】：当核心看点是"长期超额收益可持续性"时使用（ROIC 长期 > WACC 5pct 以上 + 多种护城河来源）。

## 一、稳健价值评分（100 分）

| 维度 | 权重 | 关键指标 |
|------|-----|---------|
| 财务质量 | 25 | ROE / ROIC / 毛利率 / 净利率 / 扣非稳定性 |
| 现金流质量 | 25 | 经营现金流净额 / 现金流÷净利润 / 自由现金流 / 分红支撑 |
| 资产负债表安全 | 20 | 资产负债率 / 有息负债 / 货币资金覆盖短债 / 商誉风险 / 应收存货风险 |
| 成长与行业空间 | 15 | 营收增速 / 扣非净利增速 / 行业景气 / 公司地位 |
| 估值安全边际 | 15 | PE 分位 / PB 分位 / 股息率 / FCF 收益率 |

| 指标 | 满分 | 评分规则 |
|------|------|----------|

| ROIC 均值（5年） | 5 | >15%:5, >10%:4, >7%:2, ≤7%:0 |
| 毛利率稳定性 | 5 | 5年标准差<3pp且均值>30%:5, 标准差<5pp:3, 其他:1 |
| 净利率稳定性 | 4 | 5年标准差<2pp:4, <4pp:2, 其他:1 |
| 扣非质量 | 3 | 扣非/净利润>0.9:3, >0.7:2, 其他:1 |

| 自由现金流 | 5 | 长期正且增长:5, 长期正:3, 偶尔为负:1 |

| 分红支撑 | 3 | 经营现金流支撑且FCF仍正:3, 经营现金流支撑:2, 借钱分红:0 |

## 二、捡漏型评分（100 分）

| 维度 | 权重 | 关键指标 |
|------|-----|---------|
| 安全边际 | 30 | (合理估值 / 当前价 - 1) × 估值置信度 |
| 资产质量 | 20 | NCAV / 有形资产折扣 / 资产真实性 |
| 反转催化 | 20 | 基本面是否改善？毛利率回升？现金流转正？新品/新客户？ |
| 不会立刻死 | 20 | 资产负债率 < 70% / 货币资金覆盖 1 年开支 / 无 ST 风险 |
| 行业不是夕阳 | 10 | 行业 5 年 CAGR ≥ 0% / 不在政策打压 / 不被技术替代 |

## 三、高质量成长评分（100 分）

| 维度 | 权重 | 关键指标 |
|------|-----|---------|
| 增长质量 | 30 | 营收 CAGR / 扣非 CAGR / 经营现金流 CAGR / 三者差距 |
| 单位经济 | 20 | 单 W 利润 / 单店利润 / ARPU / LTV/CAC(SaaS) |
| 护城河 | 20 | 毛利率稳定性 / 市占率趋势 / 客户粘性 |
| 资本效率 | 15 | ROIC / 资本开支强度 / 存货周转 |
| 估值合理度 | 15 | PEG / PS 分位 / DCF 隐含增长率 |

## 四、护城河型评分（100 分）

| 维度 | 权重 | 关键指标 |
|------|-----|---------|
| 护城河类型 | 25 | 5 种护城河中具备几种 / 是否互相加固 |
| ROIC 持续性 | 25 | 10 年 ROIC 均值 - WACC / 标准差 |
| 盈利质量 | 20 | 毛利率 / 净利率稳定性 / 现金流÷利润 |
| 市占率 | 15 | TOP3 / TOP5 占比 / 5 年趋势 |
| 估值 | 15 | PE / 历史分位 / DCF 隐含 ROIC 假设 |

## 评分结论分类

| 分数区间 | 分类 | 含义 |
|----------|------|------|
| 80-100 | A 类 | 值得重点研究 |
| 65-79 | B 类 | 可以加入观察池 |
| 50-64 | C 类 | 有亮点但风险较高 |
| < 50 | D 类 | 暂不建议观察 |

## 策略模板 -> 评分模式映射

| 策略模板 | 默认评分模式 | 备注 |
|---------|------------|------|
| 稳健价值型 | 稳健价值评分 | 主力模式 |
| 高质量成长型 | 高质量成长评分 | 早期/PS 估值 |
| 高股息低估值型 | 稳健价值评分（提高股息率权重） | 或视情况用护城河型 |
| 困境反转型 | 捡漏型评分 | 降低财务质量权重，提高估值边际权重 |

## 使用说明

- 评分仅用于排序和初筛，不构成买入建议。
- A/B 类公司应进一步用深度估值模式做深度研究。
- 困境反转型评分需调整权重（降低财务质量权重，提高估值边际权重）。
- 行业景气度和竞争地位需结合行业研究判断，无法完全量化。
